Usage

System Requirements

macOS (11 or later), or Windows (10 or later)

ThermoRawRead is required to process .raw files, and dotnet 8.0 is required to run ThermoRawRead.

Installation

  • Download the latest release from targetwizard.ctarn.io.
  • For Windows users, the software can be used directly without installation.
  • For macOS users, install the software using the `.pkg` release, or unzip the downloaded `.zip` file and move the `TargetWizard.app` to `/Applications`.
  • Installation (if required) should be finished in less than one minute.

Example

  • The documentations provided detailed tutorials or manuals.
  • The demo data can be downloaded from targetwizard.ctarn.io, and used to try TargetWizard.
  • For most tasks, TargetWizard can compelete various tasks within minutes.
